# 苳季or苦涩环境助手

> Magisk / KernelSU / APatch 游戏环境伪装模块

由苦涩 & 苳季联合开发，专为需要通过游戏环境检测的用户设计。

## 依赖

- [Magic Mount RS](https://github.com/5ec1cff/MagicMountRS) — CPU 白名单伪装功能需要此元模块支持

## 功能

- **防设备标记** — 游戏运行时自动挂载空目录，规避痕迹检测
- **CPU 白名单伪装** — 伪装为白名单机型 CPU 信息，概率解决检测问题
- **ColorOS 云控解锁** — 解除 OPPO/Realme 云控限制帧率
- **HMA 配置预置** — 一键导入 Hide My Applist 白名单配置
- **自动清理 ano 文件** — 游戏退出后自动清理腾讯 ano 痕迹
- **挂载空目录 Pro** — 游戏启动时短暂隐藏特定文件
- **WebUI 控制面板** — 可视化管理所有功能开关

## 支持平台

Magisk / KernelSU / APatch，Android 12+
